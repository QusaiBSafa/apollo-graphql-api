# Apollo GraphQL API

## Description
This is a simple GraphQL API built with Apollo Server and Node.js. It demonstrates how to set up a basic GraphQL server with a single resource type.

## What's Built
- A GraphQL API that allows you to query and mutate a list of books.

## How to Run
1. Clone the repository: `git clone https://github.com/yourusername/apollo-graphql-api`
2. Navigate to the project directory: `cd apollo-graphql-api`
3. Install dependencies: `npm install`
4. Start the server: `npm start`
5. Open your browser and go to `http://localhost:4000`

## API Docs
- **Query**: `books` - Get a list of all books.
- **Mutation**: `addBook(title: String!, author: String!)` - Add a new book.

## Environment Variables
No environment variables are required for this project.
